Main Content

This is the main content area of the page. It can contain articles, blog posts, or any other primary information. We are using Flexbox to structure this layout, allowing the main content and the sidebar to sit side-by-side on larger screens.

Flexbox provides a powerful way to align and distribute space among items in a container. Here, the container is set to `display: flex`, and the `flex` property on the main and aside elements determines how much of the available space they occupy.

Sidebar

This is the sidebar. It can contain related information, advertisements, navigation links, or other secondary content. On smaller screens, this sidebar will appear below the main content thanks to the media queries we've implemented.
